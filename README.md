A responsive food recipe web application built using **HTML, CSS, and JavaScript** that allows users to search for food recipes and view detailed cooking instructions, ingredients, and related information in real time.

The application fetches data from **TheMealDB public API** and dynamically displays recipes based on user input.

---

## Features
- Search recipes by food name
- Fetches real-time recipe data using API
- Displays food image, category, area, ingredients, and instructions
- User-friendly and clean UI
- Fully responsive for mobile, tablet, and desktop screens
- Error handling for invalid searches or network issues

## How It Works
1. User enters a food name in the search box.
2. Application fetches recipe data from the API.
3. Recipe details are dynamically displayed on the screen.
4. Handles loading state and error messages gracefully.
